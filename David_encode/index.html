About this Tool

Usability

This tool allows you to encode text data into an image and decode it back. It supports different color modes (RGB, Black & White, Hidden Noise) and an optional password-based obfuscation algorithm. This can be useful for:

  • Steganography: Hiding messages within seemingly innocuous images.
  • Data Transfer: Sharing small pieces of text data in an image format.
  • Secure Notes: Storing sensitive notes in an image that requires a passphrase to decode.

How-to Use

  1. Enter Text: Type or paste the text you want to encode into the "Enter text to encode..." area.
  2. Select Mode: Choose a "Color Mode"
    • RGB: Each character's byte value is mapped to R, G, or B channels.
    • Black & White: Each bit of a character is represented by a black or white pixel.
    • Hidden Noise: Embeds data into a random noise image. Requires a Passphrase for decoding.
  3. Select Algorithm: Choose "Standard" or "Password". If "Password" is selected, enter a passphrase.
  4. Encode: Click the "Encode" button. The encoded image will appear on the canvas.
  5. Download (Optional): Click "Download Image" to save the image. If "Hidden Noise" mode is used, a "Key File" will also be downloaded.
  6. Decode: To decode, upload an image using "Upload Image". Select the correct "Algorithm" (and passphrase if applicable), then click "Decode". If it's a "Hidden Noise" image, enter the key from the downloaded key file in the "Key" field.
